# 🎮 Tic-Tac-Toe AI

A modern and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**. The game features a clean user interface, player login, score tracking, sound effects, animations, and an AI opponent.

---

## 🚀 Features

- 👤 Player Login System
- 🤖 AI Opponent
- 🎯 Player vs AI Gameplay
- 🏆 Winner Detection
- 🤝 Draw Detection
- 📊 Scoreboard
- 🔄 Restart Game
- 💾 Local Storage Support
- 🎨 Responsive Design
- 🔊 Sound Effects
- ✨ Smooth Animations

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## 🎮 How to Play

1. Open **login.html**.
2. Enter your name.
3. Click **Start Game**.
4. Play against the AI.
5. Get three symbols in a row to win.
6. Click **Restart** to play again.

---

## 🧠 AI Logic

The AI automatically selects an available cell after the player's move.

Current AI features:
- Automatic move selection
- Prevents invalid moves
- Detects game over
- Provides a smooth gameplay experience
